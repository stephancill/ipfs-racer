stupid ipfs

A fast IPFS and IPNS gateway that races multiple public gateways and returns the first valid response.

Usage

# subdomain (recommended)
https://<cid>.ipfs.stupidtech.net/
https://<cid>.ipfs.stupidtech.net/path/to/resource

# path (redirects to subdomain)
https://ipfs.stupidtech.net/ipfs/<cid>/

# ipns (path only)
https://ipfs.stupidtech.net/ipns/<name>/

Example: bafybei...2u.ipfs.stupidtech.net

Gateways

Requests race these public gateways. The fastest valid response wins.

  • dweb.link
  • ipfs.io
  • gateway.pinata.cloud
